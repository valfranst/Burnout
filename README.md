# Burnout no trabalho remoto e isolamento social

Projeto que conecta análise comportamental e suporte humano a partir de uma base inicial do *Remote Work Burnout & Social Isolation (2026)*, do Kaggle, e de um cadastro online contínuo que permite ampliar os exemplos observados diariamente.

## Dataset
- Fonte primária: [Kaggle: Remote Work Burnout & Social Isolation (2026)](https://www.kaggle.com/datasets/aryanmdev/remote-work-burnout-and-social-isolation-2026?resource=download)
